Features

話す・撮る・書き起こす、すべて端末で。

録音やビデオから、その場でテキストに。さらに翻訳と AI で、伝わる言葉に。

録音して文字起こし

マイクで話すだけで、その場でテキストに。会議・取材・メモに。

動画から文字起こし

ライブラリの動画を選ぶだけ。音声を抽出してテキスト化します。

モデルを選べる

用途に合わせて Tiny / Base / Small / Large v3 を切り替え。精度と速度のバランスを選べ、各画面の上部からすぐ変更できます。

結果をそのまま翻訳Pro

文字起こしの結果を、ワンタップで約20言語に翻訳。言語パックを入れればオフラインでも、すべて端末内で。

意図転訳（AIアドバイザー）Pro

伝えたいことを入力すると、場面別の自然な言い回しを AI が3〜4案提案。Apple Intelligence と Qwen（高精度）から選べ、すべて端末内で動作。

履歴と書き出し

作った文字起こしは履歴に自動保存。検索・絞り込みで探せて、テキスト（.txt）や字幕（.srt）で書き出し。

約99言語に対応

日本語・英語・中国語ほか約99言語に対応。話す言語は自動でも判定します。

7つのテーマ

ミッドナイトなど7種類のテーマで、グラス調の UI を自分好みの見た目に。

Privacy by design

プライバシーは、設計から。

サーバーも、解析 SDK も、広告もありません。録音・動画は端末内で処理し、文字起こしは本体にのみ保存します。
